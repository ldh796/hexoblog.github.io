Embeded 프로젝트

Dec 25, 2020

플라스크를 이용한 라즈베리파이 웹페이지

프로젝트 진행중 얻은 사실 !

  • 라즈베리 웹페이지를 구성하기 위해서는 선행 사항이 있다

    • 라즈베리 파이 환경에서 python 의존성 문제로 인해 가상머신을 실행한 후에
      pip 키워드를 통해서 해당 라이브러리 (OpenCV)를 다운로드 해야한다.

요구사항 분석

  • Problem

    • 주차장에 방문하면 자리가 없는 경우가 비일비재함.

    • 유류비 및 동선 낭비를 해결하기 위한 대책 필요.

  • Solution

    • 다수의 사용자가 웹 예약이 가능해야 함

    • 차량 인증은 현장에서 번호판 인식으로 함 (추가적인 기능 구현 필요, 라즈베리 파이 구매예정)
